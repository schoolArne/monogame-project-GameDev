# controls

| action      | key         |
| ----------- | ----------- |
| quit        | escape      |
| pause       | p           |
| start game  | enter       |
| run left    | left arrow  |
| run right   | right arrow |
| jump        | up arrow    |
| fall faster | down arrow  |
